## 2. Tipografía

### Fuente Principal

_Sans-serif (System Default)_

> El veloz murciélago hindú comía feliz cardillo y kiwi.

### Fuente de Código (DX)

_`var(--font-mono)`_

```css
.ejemplo {
  font-family: var(--font-mono);
  color: var(--brand-primary);
}
```
