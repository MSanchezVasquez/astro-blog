Los Hooks revolucionaron React en la versión 16.8, permitiéndonos usar estado y ciclo de vida sin escribir clases.

## 1. useState

Es la memoria de tu componente.

```jsx
import { useState } from "react";

function Contador() {
  const [cuenta, setCuenta] = useState(0);

  return (
    <button onClick={() => setCuenta(cuenta + 1)}>Click me: {cuenta}</button>
  );
}
```

2. useEffect
   Maneja los "efectos secundarios" (peticiones API, suscripciones, timers). Reemplaza a componentDidMount, componentDidUpdate y componentWillUnmount.

```javascript
import { useEffect } from "react";

useEffect(() => {
  console.log("El componente se montó");

  return () => {
  console.log("El componente se desmontó (limpieza)");
  }
}, []); // El array vacío significa: "ejecuta solo al montar"

Nota: Es crucial entender el array de dependencias (el segundo argumento). Si lo omites, el efecto se ejecutará en cada render, lo que puede causar bucles infinitos.
```
