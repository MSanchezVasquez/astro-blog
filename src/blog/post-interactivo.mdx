¡Esto es un archivo **MDX**! Puedo escribir Markdown normal, pero también puedo importar componentes.

## 1. Componente de Preact (Interactivo)

Aquí abajo hay una "Isla" de Preact funcionando dentro del contenido:

<div
  style={{
    padding: "20px",
    backgroundColor: "var(--color-slate-100)",
    borderRadius: "8px",
  }}
>
  <Greeting client:load messages={["Hola", "Qué tal", "Bienvenido a MDX"]} />
</div>

## 2. Componente de Astro (Estático)

También puedo reutilizar componentes de UI de Astro, como el botón de tema:

<div style={{ marginTop: "20px" }}>
  <ThemeIcon />
</div>

> **Nota:** Fíjate cómo puedo mezclar sintaxis Markdown con etiquetas HTML/JSX directas.
